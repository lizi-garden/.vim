# Vim配置

参考[https://github.com/skywind3000/vim-init](https://github.com/skywind3000/vim-init)对原先的配置重构。

## 安装
克隆到`~/.vim/`内：
```bash
cd ~/.vim
git clone https://github.com/skywind3000/vim-init.git
```

在.vimrc里添加
```vimscript
source ~/.vim/vim-init/init.vim
```

## 功能
- C/C++代码补全
- 漂亮的主题
